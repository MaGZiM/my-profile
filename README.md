# Проект "Сайт-портфолио"

> Сайт состоит из 4х страниц и админки.
 1. Страница авторизации
 2. Обо мне
 3. Мои работы
 4. Страница блога
 
> Для создания сайта используются след. технологии:
 1. HTML
 2. CSS
 3. Pug
 4. Sass
 5. NPM
 6. Сборка работает на gulp версии 4.0.
